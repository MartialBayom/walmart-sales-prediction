#  Walmart — Prédiction des ventes hebdomadaires

##  Objectif
Construire un modèle ML pour prédire les ventes hebdomadaires 
de 45 magasins Walmart en intégrant des variables externes 
(fériés, météo, économie).

##  Dataset
- 45 magasins · 3 fichiers CSV (stores, features, sales)
- Variables : température, carburant, chômage, événements fériés

##  Méthodologie
1. Analyse exploratoire (EDA) + visualisations
2. Preprocessing : encodage, normalisation, gestion des NaN
3. Modèle baseline : Régression Linéaire
4. Modèles régularisés : Ridge & Lasso
5. Optimisation : GridSearchCV
6. Évaluation : RMSE / R²
